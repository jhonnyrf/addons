# Diseño Visual Wigo

Módulo que mejora significativamente la experiencia visual del navbar y sidebar de aplicaciones en Odoo 19.

## Características

✨ **Mejoras visuales del sidebar de apps:**

- Diseño elegante y moderno con gradientes suaves
- Iconos destacados con sombras mejoradas
- Transiciones y animaciones fluidas
- Efectos hover interactivos
- Indicadores visuales de la aplicación activa

🎨 **Estilos mejorados:**

- Paleta de colores coherente y moderna
- Tipografía mejorada
- Espaciado optimizado para legibilidad
- Sombras sutiles para profundidad
- Bordes redondeados y transiciones suaves

📱 **Responsividad:**

- Perfecto en pantallas pequeñas (móvil)
- Mejorado en pantallas medianas (tablet)
- Elegante en pantallas grandes (desktop)
- Diseño adaptable a cualquier resolución

🔍 **Interactividad:**

- Efectos ripple al hacer clic
- Animaciones de transición suaves
- Estados hover intuitivos
- Scroll personalizado en el menú

## Instalación

1. Coloca el módulo en tu carpeta de addons
2. Actualiza la lista de módulos
3. Busca e instala "Diseño Visual Wigo"
4. Recarga la página para ver los cambios

## Requisitos

- Odoo 19.0
- Módulo base: `web`

## Características técnicas

- Assets SCSS compilado automáticamente
- JavaScript sin dependencias externas
- Compatible con temas Odoo estándar
- Sin cambios en la estructura del core
