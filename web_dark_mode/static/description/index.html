Wigo Fast Dark Mode

Este modulo fue hecho por desarrolladores de Wigo Fast para habilitar modo oscuro en los modulos de Wigo Fast sobre Odoo.

Que hace este modulo

  • Permite cambiar entre modo claro y modo oscuro desde el menu de usuario.
  • Guarda la preferencia del usuario para mantener la experiencia visual.
  • Aplica estilos oscuros para mejorar contraste y lectura.

Uso interno

Modulo orientado a implementaciones y soporte interno de Wigo Fast.
