===================
Wigo Fast Dark Mode
===================

Descripcion
===========

Este modulo fue desarrollado por desarrolladores de Wigo Fast para habilitar
modo oscuro en los modulos internos de Wigo Fast sobre Odoo.

El cambio de tema se controla desde el menu de usuario (arriba a la derecha)
y permite alternar entre tema claro y oscuro.

Objetivo
========

El objetivo del modulo es mejorar la experiencia visual en trabajo diario,
especialmente en sesiones largas, aplicando estilos oscuros compatibles con
la interfaz web de Odoo.

Funcionalidad
=============

- Activacion de modo oscuro desde el menu de usuario.
- Persistencia de preferencia por usuario.
- Compatibilidad con renderizado web y assets de tema oscuro.
- Sobrescrituras visuales para mejorar contraste en vistas clave.

Soporte Interno
===============

Este modulo fue hecho para uso y soporte interno de Wigo Fast.

Autores
